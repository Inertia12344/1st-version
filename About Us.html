    General

    This is a site where consumers can browse and purchase private islands owned by a private company. The Home page displays the qualities and promises made by the company on the high standard of private islands they provide. The Luxury, Exotic and Favourites pages show the three different categories of islands on offer. Each page has a similar layout, displaying 4 different islands for sale, which includes a name label and a starting price. Additionally it provides a section informing interested customers to email isles@inertia.com for requests, with a disclaimer that there are no guarantees and prices are subject to change. The About page contains a table showing past purchases made, indicating the island name, price and customer review. It then contains the rest of About this Site. The footer of the website offers links to the company’s policies, buying and renting, information here at Isles of Inertia and membership details. In addition to the site, a search bar was added. A favicon showing the letter “i” was used.

    Bonus Marks

    One feature of my site that deserves bonus marks is the search bar function in the navigation bar that allows you to search information on the website. The actual searching feature isn’t there because it would require code from Javascript and PHP, but the search bar is included. Additionally all of the CSS code has been properly commented and organised as part of the code. There is evidently an overall consistency in my website, I used a 3 colour palette: Orange (228, 148, 0), Dark Navy (1, 1, 61) and plain white. I used the Aboreto font primarily in my website, downloaded from Google Fonts, and the Font Family: ‘Lucinda Sans’, ‘Lucida Sans Regular” etc. By using a consistent font and colour scheme, my site has an evident overall appearance and consistency. Furthermore, all the images in my website are responsive. I have used lists throughout my whole website, as seen in the navigation bar and for the list of references. In addition, I also linked sections of my About page using list and hyperlinks to enhance the experience and use of the website. These are all things about my site that I believe deserve bonus marks.
